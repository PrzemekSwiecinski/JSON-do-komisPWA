# Strona komisu samochodowego PWA / PWA car dealership website

(PL) Responsywna strona internetowa typu PWA dla komisu samochodowego, która pobiera oferty pojazdów z zewnętrznego pliku JSON i umożliwia ich przeglądanie oraz sortowanie. 
Dzięki technologii Service Worker aplikacja działa w pełni offline i pozwala użytkownikom na jej bezpośrednią instalację na telefonie lub komputerze.

(EN) A responsive PWA website for a used car dealership that retrieves vehicle offers from an external JSON file and allows users to browse and sort them.
Thanks to Service Worker technology, the app works completely offline and allows users to install it directly on their phone or computer.
